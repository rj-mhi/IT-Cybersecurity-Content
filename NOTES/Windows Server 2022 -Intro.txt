Key Features of Windows Server
-----------------------------------------
	Active Directory: Manages permissions and access to resources across the network.
	Server Manager: A centralized dashboard to manage multiple servers and roles.
	Hyper-V: A hypervisor for creating and managing virtual machines.
	File and Storage Services: Handles data storage, including file sharing and access management.
	Security Features: Includes features like BitLocker, Windows Defender, and other network security tools.
	Remote Desktop Services: Allows multiple users to access Windows-based applications and desktops remotely.
